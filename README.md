# CT Heat Pump Incentive Scenario Explorer

Companion tool to the draft white paper *Not All Heat Pump Conversions Are
Equal: Fuel-Differentiated Incentives and the Cost of Decarbonizing
Connecticut Homes* (Scruggs, Ouimet, and Bonitz, April 2026).

The app lets a user define an arbitrary state rebate structure — per-fuel rate
(either `$/cap-ton` or `% of HP cost`) with optional dollar caps, plus an
optional federal layer (flat or HEAR stacking) — and see projected statewide
5-year outcomes alongside the eight preset scenarios reported in Table 6 of
the paper.

## What the app reports

For each scenario, the same five metrics as paper Table 6:

- 5-year conversions (total, oil+propane, gas)
- Annual CO₂ savings (tons/year)
- Total program cost (5 years)
- Mean rebate per conversion
- Program $/ton CO₂ (20-year equipment life)
- Total resource $/ton CO₂ (program + private premium, 20-year)

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

Opens at http://localhost:8501.

## Deploy to Streamlit Community Cloud (free)

1. Push this directory to a new GitHub repo (public or private — both work
   on the free tier for personal accounts).
2. Visit https://share.streamlit.io and sign in with GitHub.
3. Click **New app**, select the repo, set main file to `app.py`.
4. Click **Deploy**. It reads `requirements.txt` and boots in a minute or two.

Future updates: push a commit, the app redeploys automatically.

## Updating the data / coefficients

The committed `app_data/` files were generated by `prepare_data.py` running
against the paper's source files in the parent `eec/` Dropbox folder
(`eec25_analysis_ready.csv` and `new_20k_address.dta`). **Those raw files are
not in this repo**. If you re-run the paper's analysis and want the app to
pick up the new numbers, regenerate the data locally:

```bash
python prepare_data.py
git add app_data/
git commit -m "Refresh app data to match updated analysis"
git push
```

Streamlit Cloud will redeploy automatically.

## Caveats

The logit was fit on survey respondents with heating systems aged 15+ years
at prevailing 2025 fuel and electricity prices. Extrapolating to very
different price regimes (e.g., large oil-price shocks) should be treated as
indicative rather than forecast. A residual cap of 85% is applied to the
per-home choice probability to reflect non-adoption for reasons outside the
economic gap (aesthetics, landlord/tenant splits, electrical capacity,
installer availability). Equipment-cost and fuel-use formulas mirror the
Qualtrics survey instrument; alternative cost curves shift absolute levels
but rarely the ordering between scenarios.
